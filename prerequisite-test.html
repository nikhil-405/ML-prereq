Question 1

To multiply a (n, k) matrix by a (k, m) matrix, we ewill need a total of n * m * k multiplications and n * m * (k-1) number of addition operations,
The first matrix has n rows and each row has k entries, the second matrix has k rows with each row having m entries,
so when we try to multiply the first row of the first matrix, we will have to multiply (k, k) elements to each other and then add them.
We will have to repeat this for all m columns of the second matrix for each of the n rows in the first matrix

Question 3

If we were to use the given method to find the median of an array of length n,
the time complexity would be O(n**2). This is because we will need to loop over the length of the list n//2 times.

A better approach would be to sort the list (O(n * log n)) and assess its n//2 th element (O(1)). This would result in a time complexity O(n * log n).

In[75]:
def n_square_method(l, count = 0):
    highest = []
    while count <= len(l)//2:
        max_ = -float('inf')
        for i in range(len(l)):
            if (l[i] not in highest) and (l[i] > max_):
                max_ = l[i]
        count += 1
        highest.append(max_)
    return highest[-1]

def arg_minimize(arr):
    arr_flatten = list(arr.flatten())
    min_elem = float('inf')
    min_index = 0
    for i in range(len(arr_flatten)):
        if arr_flatten[i] < min_elem: 
            min_index = i
            min_elem = arr_flatten[i]
    return min_index
